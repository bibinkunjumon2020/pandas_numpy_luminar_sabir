    A convenient alias for None, useful for indexing arrays. 
 
    Examples 
    -------- 
    >>> newaxis is None 
    True 
    >>> x = np.arange(3) 
    >>> x 
    array([0, 1, 2]) 
    >>> x[:, newaxis] 
    array([[0],

    >>> x[:, newaxis, newaxis] 
    array([[[0]], 
    [[1]],

    >>> x[:, newaxis] * x 
    array([[0, 0, 0], 
    [0, 1, 2], 
    [0, 2, 4]]) 
 
    Outer product, same as ``outer(x, y)``: 
 
    >>> y = np.arange(3, 6) 
    >>> x[:, newaxis] * y 
    array([[ 0,  0,  0], 
    [ 3,  4,  5], 
    [ 6,  8, 10]]) 
 
    ``x[newaxis, :]`` is equivalent to ``x[newaxis]`` and ``x[None]``: 
 
    >>> x[newaxis, :].shape 
    (1, 3) 
    >>> x[newaxis].shape 
    (1, 3) 
    >>> x[None].shape 
    (1, 3) 
    >>> x[:, newaxis].shape
